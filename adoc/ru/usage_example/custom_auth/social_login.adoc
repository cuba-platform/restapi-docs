==== Social Login в REST API

Механизм авторизации через социальные сети, или social login, также можно использовать в REST API. Исходный код приложения, описанного в этом примере, доступен на https://github.com/cuba-platform/sample-social-login-cuba7[GitHub], а сам пример описан в разделе {manual}/social_login.html[social login]. Ниже приведены ключевые моменты этого примера, позволяющие получить OAuth-токен через аккаунт Facebook.

. Создайте пакет `restapi` в корневом каталоге модуля *web* и поместите в него собственный контроллер Spring MVC. Контроллер должен содержать два основных метода: `get()`, возвращающий `ResponseEntity`, и `login()`, в котором мы будем получать OAuth-токен.
+
--
